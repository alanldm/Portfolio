LASC - Payload

The Latin American Space Challenge (LASC) is an international aerospace development event. In the 2020 edition, I participated as a member of the electronics sector of the Potiguar Rocket Design (PRD). My contribution was to develop a payload that had the function of detecting fire outbreaks. For this, the device, built in a modular way, had a camera (OV7670), an ESP32 and a memory card. In addition, it had some other components such as a gyroscope (MPU6050) and an altimeter (BMP280). The programming of the equipment was done using the Arduino IDE.

H100

The H100 consisted of a rocket that reached 100 meters as its maximum apogee. Due to the small size of the rocket, its electronics had to be reduced as well. Thus, the H100's avionics consisted only of an altimeter (BMP280), an ATTiny85, a memory module (AT24C256) and a lithium battery.

H300

Similar to the H100, the H300 was a rocket that reached a maximum of 300 meters and was developed in the year 2021. The great thing about this avionics was that it communicated with the recovery system, activating the circuit that released the rocket's parachute. In addition, the H300's avionics included a gyroscope (MPU6050), an altimeter (BMP280), a memory module, lithium batteries, and an ATmega328p. However, due to a power supply problem with the boost converter, the design had to be re-adapted.

OBSAT

The Brazilian Satellite Olympics (OBSAT) is a national competition that aims to foster practical and theoretical experiences with small satellites. In the 2021 edition, the team led by me won a nanosatellite after passing the first phase of the tournament. Control Systems

Besides the purely theoretical concepts learned in Control I discipline, I was able to develop two main projects: a code to map the roots of a function using Newton's method and a controller for a hit-the-target game. This last project had the objective of controlling a character, via API, from the information received by this channel. Since each phase consisted of a different vector field that behaved differently for each type of input, it was necessary to create a "predictor" code, where numerous inputs and trajectories that the character would do were simulated and the most favorable one to hit the target was chosen.

Computational Methods

For the computational methods discipline, I was able to do 3 projects: a code to calculate the voltage on a triangular plate using Laplace's numerical solution, a code to find the roots of a function using the Runge-Kutta method of 6th order and a code to solve integrals, which used the trapezoidal rule and Newton-Cotes. The second project was the most important, because in it I was able to translate the code made in Python to other 3 languages C++, Julia and Matlab.
